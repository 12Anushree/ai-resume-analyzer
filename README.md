## 🌟 About the Project

**AI Resume Analyzer Dashboard** is a simple web app that helps users compare their resume with a job description.

It gives:
- ✅ Resume match score
- ✅ Missing keywords
- ✅ Skill gap analysis
- ✅ AI suggestions to improve the resume
- ✅ Easy dashboard view with clean UI

---

## 🚀 Features

- 📄 Upload resume in **PDF / DOCX / TXT**
- 📝 Paste or type a **job description**
- 🤖 AI compares resume with job requirements
- 📊 Shows **match score**
- 🔍 Highlights **matched skills** and **missing skills**
- 💡 Gives **resume improvement suggestions**
- 📈 Beautiful dashboard style interface
- ⚡ Fast and easy to use
